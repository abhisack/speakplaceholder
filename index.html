'speakplaceholder.js' is a tiny library which makes your inputs speak their placeholders on being focussed. Let your forms speak!

How to use

Get it

clone from GitHub-

            
            
  git clone https://github.com/abhisack/speakplaceholder.git

Or fetch with npm

             
            
  npm i speakplaceholder

Include file

Simply include speakplaceholder.js in your page

      
      
  <script src="../speakplaceholder.js></script>

Get Your Input Elements Ready

speakplaceholder.js speaks the placeholders of all the input elements that have a placeholder attribute with some value and data-spl attribute set to true.

     
     
  <input placeholder="Enter your email" data-spl="true" type="text">

Set Pitch, Volume & Rate

You can set an input element specific pitch, volume and speech-rate using data-spl-pitch, data-spl-volume & data-spl-rate attributes respectively. 

         
         
  <input placeholder="Enter your name"
   data-spl="true" data-spl-pitch="0.5" 
   data-spl-volume="0.5" data-spl-rate="1.5" 
   type="text"/>

Browser Support

speakplaceholder.js works on all modern browsers that support Speech Synthesis API. Check it's browser support on caniuse. However, this browser is supported.
